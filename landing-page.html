Detecting differences in code

Rather than only highlight visual differences, SiteDiff compares different versions of code highlighting differences and allowing developers and QA testers to understand and correct code regressions before releasing.

Sitediff emphasizes automation

SiteDiff automates a big portion of the QA process by automatically crawling the whole site to look for regressions. It applies a rule based refinement process to highlight breaking changes and differentiate them from expected minor variations.
